# Logo Assets

To add the Adaptive.live logo to this documentation site:

1. Place logo files in this directory:
   - `adaptive-logo.svg` - Vector logo (recommended)
   - `adaptive-logo.png` - PNG logo (fallback)

2. The logo is already referenced in the main layout

## Logo Usage

The logo appears in:
- Site header
- Navigation bar
- Footer

## Official Logo

Download the official Adaptive.live logo from:
- https://adaptive.live (website)
- Contact Adaptive for brand assets
